# =============================================================================
# Lead Gen Multi-Agent — Python Dependencies
# =============================================================================
#
# INSTALL ORDER — run these after `pip install -r requirements.txt`:
#
#   1. playwright install firefox
#        Downloads the Playwright Firefox binary (used by playwright-stealth,
#        browser-use, and as a fallback; Camoufox ships its own Firefox).
#
#   2. python -m camoufox fetch
#        Downloads the patched anti-fingerprint Firefox binary for Camoufox.
#        Must be run separately — pip install alone is NOT enough.
#
#   3. crawl4ai-setup   (optional — only if Crawl4AI fails on first use)
#        Some systems need this to install Playwright Chromium for Crawl4AI.
#
# =============================================================================
